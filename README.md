# Restaurant Menu - Bella's Italian Kitchen

A beautiful, responsive restaurant menu page built with HTML and CSS, featuring elegant typography and layouts.

## Project Overview

This project is a static restaurant menu website showcasing authentic Italian cuisine.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Elegant color scheme inspired by vintage floral patterns
- Organized menu sections: Appeizers, Main Courses, Desserts, and Drinks

## Technologies Used

- HTML5
- CSS3

## Getting Started

### View Live Demo

[Live Demo](https://ssyevuo.github.io/restaurant-menu/)

### Run Locally

1. Clone the repository:
```bash
git clone https://github.com/ssyevuo/restaurant-menu.git
```

2. Navigate to the project directory:
```bash
cd repository-name
```

3. Open `index.html` in your browser:
```bash
# On Mac
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```
